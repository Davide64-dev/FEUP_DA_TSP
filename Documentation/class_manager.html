double Manager::computeDelta ( const std::vector< int > &  tour,
int  i,
int  j,
bool  isFullyConnected = true 
)
private

Computes the change in the total distance of a tour if a segment between nodes i and j is reversed.

This function calculates the change (delta) in the total distance of a tour if the segment between the indices i and j is reversed. It is used in optimization algorithms such as 2-opt for the Traveling Salesman Problem (TSP) to determine if reversing a segment reduces the total tour distance.

Parameters
tourA vector of integers representing the sequence of nodes in the current tour.
iThe index of the first node in the tour where a modification is being considered.
jThe index of the second node in the tour where a modification is being considered.
isFullyConnectedA boolean flag indicating whether the graph is fully connected or not. This parameter might influence the behavior of the distance function.
Returns
The change (delta) in the total distance if the segment between i and j is reversed.
Complexity
  • Time: O(1)
  • Space: O(1)
